## 🚀 Quick Start

### Option 1: Automated Setup (Recommended)

**Windows:**
```bash
setup.bat
```

**Linux/Mac:**
```bash
chmod +x setup.sh
./setup.sh
```

### Option 2: Manual Setup

```bash
# 1. Install dependencies
npm install

# 2. Set up Convex backend
npx convex dev

# 3. Start development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## 📋 What's Built

### ✅ Public Pages
- **Homepage** - Featured content, events, stats, timeline preview
- **About** - Organization info, ideology, structure, Dr. Ambedkar bio
- **History** - Interactive timeline (1924-present) with era filtering
- **Articles** - Repository with search, filters, categories, tags
- **Events** - Calendar with registration system
- **Join** - Multi-step volunteer registration
- **Gallery** - Media gallery (photos, videos, documents)
- **News** - Latest announcements

### ✅ Member Portal
- **Dashboard** - Overview, stats, quick actions, registered events
- **Profile** - Edit personal information, language preference
- **Membership Card** - Digital card with QR code verification

### ✅ Admin Panel
- **Dashboard** - Stats, quick actions, recent content
- **Articles CMS** - Create, edit, publish, archive articles
- **AI Content Generator** - Auto-generate articles with AI
- **Members Management** - Approve/reject applications
- **Events Management** - Create and manage events
- **News Management** - Post announcements

### ✅ AI Features
- **Chatbot** - AI assistant on all pages (bottom-right corner)
- **Content Generation** - Auto-write articles from prompts
- **Translation** - English ↔ Hindi support

### ✅ Multilingual Support
- English and Hindi throughout the platform
- Language toggle in navigation
- Member language preferences

## 🔧 Environment Setup

### Required Services

1. **Convex** (Backend)
   - Free tier available
   - [Sign up at convex.dev](https://convex.dev)
   - Run `npx convex dev` to set up

2. **Clerk** (Authentication)
   - Free tier available
   - [Sign up at clerk.com](https://clerk.com)
   - Add keys to `.env.local`

3. **Anthropic API** (AI Features - Optional)
   - [Get API key](https://console.anthropic.com)
   - Add to `.env.local`

### Environment Variables

Create `.env.local`:

```env
# Convex (auto-generated by npx convex dev)
NEXT_PUBLIC_CONVEX_URL=https://your-deployment.convex.cloud

# Clerk (from clerk.com dashboard)
NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=pk_test_...
CLERK_SECRET_KEY=sk_test_...

# Anthropic API (optional, for AI features)
ANTHROPIC_API_KEY=sk-ant-...
```

## 🆘 Troubleshooting

### Convex Connection Error
```
Error: Could not find public function for 'articles:list'
```
**Solution:** Run `npx convex dev` to deploy Convex functions

### Clerk Auth Error
```
Error: The publishableKey passed to Clerk is invalid
```
**Solution:** Get correct keys from [clerk.com](https://clerk.com)

### Build Fails
```
Module not found: convex/_generated/api
```
**Solution:** Run `npx convex dev` to generate types
